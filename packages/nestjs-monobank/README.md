# NestJS Monobank

Цей модуль дозволяє швидко інтегрувати платіжну систему Monobank у ваші додатки на NestJS. Ви зможете створювати платежі, відстежувати транзакції, отримувати детальну інформацію про мерчанти та виконувати інші дії з фінансами без складнощів.

Повний опис можливостей, приклади використання та документацію можна знайти на [nestjs-monobank.vercel.app](https://nestjs-monobank.vercel.app).
