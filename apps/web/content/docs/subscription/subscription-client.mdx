У цьому розділі ми розглянемо, як отримати список регулярних підписок клієнта. Ми покажемо, як підготувати параметри запиту, викликати API Monobank та обробити відповідь від сервісу.

### Підготовка параметрів запиту

Для отримання списку підписок необхідно підготувати об'єкт `ClientSubscriptionsRequest`. За допомогою нього можна фільтрувати підписки за статусом, періодом створення та використовувати пагінацію.

```typescript
import { type ClientSubscriptionsRequest } from 'nestjs-monobank';

const query: ClientSubscriptionsRequest = {
     status: 'active',
     limit: 20,
     page: 1,
     dateFrom: '2024-06-01T00:00:00+03:00',
     dateTo: '2024-06-30T23:59:59+03:00'
};
```

### Виклик методу для отримання підписок

Після підготовки параметрів можна скористатися методом `getClientSubscriptions` із сервісу `MonobankService`.

```typescript
import { Injectable } from '@nestjs/common';
import {
     type ClientSubscriptionsRequest,
     MonobankService
} from 'nestjs-monobank';

@Injectable()
export class SubscriptionService {
     constructor(
          private readonly monobankService: MonobankService
     ) {}

     async getSubscriptions() {
          const query: ClientSubscriptionsRequest = {
               status: 'active',
               limit: 20,
               page: 1,
               dateFrom: '2024-06-01T00:00:00+03:00',
               dateTo: '2024-06-30T23:59:59+03:00'
          };

          const subscriptions =
               await this.monobankService.subscription.getClientSubscriptions(
                    query
               );

          return subscriptions;
     }
}
```

### Обробка відповіді від API

У відповіді API повертає список підписок та інформацію про пагінацію.

```json
{
     "items": [
          {
               "subscriptionId": "s2_AbrCdXyZ13",
               "amount": 4200,
               "interval": "1m",
               "startDate": "2024-06-01T14:15:22Z",
               "created": "2024-05-30T12:10:05Z",
               "nextChargeDate": "2024-07-01T14:15:22Z",
               "endDate": "2024-09-01T14:15:22Z",
               "status": "active"
          }
     ],
     "pagination": {
          "totalItems": 5,
          "itemsPerPage": 20,
          "currentPage": 1,
          "totalPages": 1
     }
}
```

Масив `items` містить інформацію про кожну підписку, включаючи її ідентифікатор, суму списання, інтервал, дату початку, дату наступного списання та поточний статус. Об'єкт `pagination` містить інформацію про кількість елементів, сторінок та поточну сторінку, що дозволяє реалізувати посторінкову навігацію.
